# WangsVue Components

Welcome to the comprehensive WangsVue component library. Below you'll find all available components organized by category for easy navigation and discovery.

## Quick Start

All components can be imported from the main package:

```ts
import { Button, Dialog, DataTable } from '@fewangsit/wangsvue';
```

## Component Categories

### Form Components

Interactive form elements for data collection and validation.

- [Button](/components/button) - Enhanced button with icons and styling variants
- [InputText](/components/inputtext) - Text input with validation and formatting
- [InputNumber](/components/inputnumber) - Numeric input with validation
- [InputCurrency](/components/inputcurrency) - Currency formatted input
- [InputEmail](/components/inputemail) - Email input with validation
- [InputPassword](/components/inputpassword) - Password input with strength indicators
- [InputPhoneNumber](/components/inputphonenumber) - Phone number input with formatting
- [InputRangeNumber](/components/inputrangenumber) - Range slider for numeric values
- [InputURL](/components/inputurl) - URL input with validation
- [InputBadge](/components/inputbadge) - Input with badge display
- [TextArea](/components/textarea) - Multi-line text input
- [Checkbox](/components/checkbox) - Checkbox selection component
- [ToggleSwitch](/components/toggleswitch) - On/off toggle switch
- [MultiSelect](/components/multiselect) - Multi-select dropdown with search
- [Form](/components/form) - Form wrapper with validation and submission
- [FileUpload](/components/fileupload) - File upload with drag and drop
