## Basic Usage

```vue
<script setup lang="ts">
import { Button } from '@fewangsit/wangsvue';
</script>

<template>
  <Button label="Click Me" />
</template>
```

## Props

### Core Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `label` | `string` | - | Text displayed on the button |
| `icon` | `string` | - | Icon name to display |
| `loading` | `boolean` | `false` | Shows loading state |
| `disabled` | `boolean` | `false` | Disables the button |

### Styling Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `severity` | `'secondary' \| 'success' \| 'info' \| 'warning' \| 'danger' \| 'help' \| 'contrast'` | `'secondary'` | Button color theme |
| `size` | `'small' \| 'large'` | - | Button size variant |
| `raised` | `boolean` | `false` | Adds shadow/elevation |
| `rounded` | `boolean` | `false` | Makes button fully rounded |
| `text` | `boolean` | `false` | Text-only style (no background) |
| `outlined` | `boolean` | `false` | Outlined style |
| `plain` | `boolean` | `false` | Plain style |
| `link` | `boolean` | `false` | Link-like appearance |

### Icon Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `icon-pos` | `'left' \| 'right' \| 'top' \| 'bottom'` | `'left'` | Icon position relative to label |

### Badge Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `badge` | `string` | - | Badge text/number |
| `badge-severity` | `'success' \| 'info' \| 'warning' \| 'danger'` | - | Badge color theme |

### Tooltip Props

| Prop | Type | Default | Description |
|------|------|---------|-------------|
| `tooltip` | `string` | - | Tooltip text |
| `tooltip-pos` | `'top' \| 'bottom' \| 'left' \| 'right'` | `'bottom'` | Tooltip position |

## Events

| Event | Payload | Description |
|-------|---------|-------------|
| `@click` | `Event` | Fired when button is clicked |

## Slots

| Slot | Description |
|------|-------------|
| `default` | Custom button content (overrides label) |
| `icon` | Custom icon content |
| `loadingicon` | Custom loading icon |

### Interactive Examples

```vue
<script setup lang="ts">
import { ref } from 'vue';
import { Button } from '@fewangsit/wangsvue';

const isLoading = ref(false);
const clickCount = ref(0);

const toggleLoading = () => {
  isLoading.value = !isLoading.value;
  if (isLoading.value) {
    setTimeout(() => {
      isLoading.value = false;
    }, 3000);
  }
};

const incrementCounter = () => {
  clickCount.value++;
};
</script>

<template>
  <!-- Toggle loading -->
  <Button
    :label="isLoading ? 'Loading...' : 'Start Process'"
    :loading="isLoading"
    @click="toggleLoading"
  />

  <!-- Counter button -->
  <Button
    :badge="clickCount > 0 ? clickCount.toString() : undefined"
    :label="`Clicked ${clickCount} times`"
    @click="incrementCounter"
    badge-severity="info"
    icon="add"
  />
</template>
```

## Accessibility

The Button component includes built-in accessibility features:

- Proper ARIA attributes
- Keyboard navigation support
- Focus management
- Screen reader compatibility
- Tooltip accessibility

## Best Practices

1. **Use appropriate severity**: Choose severity based on the action's importance and context
2. **Provide tooltips for icon-only buttons**: Always include tooltips for better UX
3. **Handle loading states**: Use the loading prop for async operations
4. **Consider button hierarchy**: Use different styles to establish visual hierarchy
5. **Keep labels concise**: Use clear, action-oriented labels
6. **Use badges sparingly**: Only for important notifications or counts
